# UEM Scripts

A collection of scripts, sensors, and tools for Omnissa Workspace ONE UEM.

This repository contains resources for managing and automating device management tasks across macOS and Windows platforms, including detection and remediation scripts.

## Contents

| Folder | Description |
|---|---|
| [Openclaw](Openclaw/) | Detection and removal scripts for the OpenClaw supply-chain compromise. Includes PowerShell (Windows) and Bash (macOS) variants for both detection and remediation, with Workspace ONE Sensor and Script integration support. |
